INSTALLATION OF FRAMEWORK:

  1. Install python from python.org and verify using python3 --version.
  2. Install pip using pip installation guide and verify using pip --version.
  3. Make a directory for the flask project mkdir problem3-flask.
  4. Move into the directory cd problem3-flask.
  5. Install flask using pip install flask.
  6. Create the folder/file structure and code (see in "HOW I BUILT THE WEBSITE").
  7. Run program using flask run.

HOW I BUILT THE WEBSITE:

  1. Developed the folder/file structure of my website.
    • Created a templates folder for my HTML files: base.html & development.html.
    • Created a static folder for CSS stylesheet and assets: styles.css.
    • Created an app.py used for the Flask framework backend.
    
                        
                        problem3-flask/
                        |── app.py              
                        |── templates/           
                        |   └── base.html    
                        |   └── development.html          
                        |   ── static/              
                            └── styles.css
  2. Set up the Flask backend in app.py:
    • Imported Flask and render_template.
    • Defined the routes for the Home and Development pages.
    • Rendered HTML templates using render_template().
    

                        from flask import Flask, render_template

                        app = Flask(__name__)

                        @app.route('/')
                        def home():
                            return render_template('base.html')

                        @app.route('/development')
                        def development():
                            return render_template('development.html')

                        if __name__ == '__main__':
                            app.run(debug=True)
  3. Designed the home page HTML code in base.html:
    • Added a consistent header (includes the nav bar), and footer.
    • Used the { % block content % } and { % block title % } for content specfic pages.
    • Added html code to create a landing page.
    

                        For example, by placing { % block content % }{ % endblock % } code 
                        into the bas.html file, any other file that extends base.html can 
                        place its unique code within these braces and it will be rendered!
  4. Designed the Development page and added CSS:
    • Extended the base.html template to get the header and footer for the development.html.
    • Added additional code to the development.html body for its page specific content.
    • Applied CSS styling to the Home and Development page using styles.css.


  5. Deployed the webpage:
    • Ran the Flask app using flask run.
    • Accessed the website at http://127.0.0.1:5000.
    • Debugged errors and fixed styling issues.
    • Uploaded the webpage to a hosting platform.

CHALLENGES

  • Template Errors: Missing template files caused rendering errors when trying to create new routes. To solve this issue, I ensured all files were in the templates folder and routed correctly (e.g. proper spelling).
  • CSS Stylings: Some of the styles I created didn't apply as I intended them to. To solve this issue, I debugged using different styles, noticing I had conflicting style rules.
  • Folder Styling: In one portion of this project I wanted to display a folder/file structure, which would be challanging to do. To solve this issue I used the pre tag.
  • Hosting Difficulties: Setting up a was something that was new to me and was timeconsuming. To solve this issue, I watched youtube videos to understand how to host the webpage.
